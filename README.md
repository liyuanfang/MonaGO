# MonaGO


## Window

(1) Install python2.7  
(2) Install dependent packages using **python setup.py**  
(3) Click cmd.cmd in the server folder  
(4) Access the webpage at [http://localhost:80](http://localhost:80)  

## Linux/Mac

(1) Install python2.7  
(2) Install dependent packages using **python setup.py**  
(3) run **python server.py**  
(4) Access the webpage at [http://localhost:80](http://localhost:80)
